### DNS Setup Steps

**⚠️ PREREQUISITE: Register `worldwidebro.com` First**

**DNS check shows `worldwidebro.com` is NOT currently registered.** You must register it before any of the following steps will work.

1. **Register the domain** → https://www.cloudflare.com or https://www.namecheap.com (~$10-15/yr)
2. **Use Cloudflare for DNS** — Free tier, fast propagation, automatic SSL
3. Vercel Hobby plan supports custom domains (1 per project); Pro plan for unlimited subdomains

**Step-by-step:**

```bash
# 1. Add domain to Vercel project (pitch-kit)
vercel domains add demo.worldwidebro.com

# 2. Vercel will give you DNS records to add:
#    Type: CNAME
#    Name: demo
#    Value: cname.vercel-dns.com

# 3. Add these DNS records at your domain registrar:
#    A Record:     @          → 76.76.21.21     (Vercel IP)
#    CNAME:        www        → cname.vercel-dns.com
#    CNAME:        demo       → cname.vercel-dns.com
#    CNAME:        hub        → cname.vercel-dns.com
#    CNAME:        upnext     → cname.vercel-dns.com
#    CNAME:        api        → cname.vercel-dns.com  (or point to Convex)

# 4. Update vercel.json in each project to set the domain
```

**For each Vercel project:**

```bash
# Pitch Kit → demo.worldwidebro.com
cd pitch-kit && vercel domains add demo.worldwidebro.com

# Venture Hub → hub.worldwidebro.com
cd venture-hub && vercel domains add hub.worldwidebro.com

# Up Next Web → upnext.worldwidebro.com
cd Upnext/bw-001-up-next-web && vercel domains add upnext.worldwidebro.com
```
